This page includes intutive methods for solving a tetrahedron, hexahedron (cube), octahedron, or dodecahedron twisty-puzzle of any size. Examples of these puzzles are Pyraminx, Rubik's cube, FTO (face turning octahedron), and Megaminx. These are not fast solutions. If you want fast solutions, look elsewhere.

Additional resources for working with virtual twisty-puzzles are also included.

twiddler A single-file app for manipulating virtual twisty-puzzles. (e.g. pyraminx, cube, FTO, megaminx). This is intended to work in the browser of any mobile device (using touch input), but should also work in any desktop browser (using mouse input). The twedra.js library is embedded in the file so no HTTP server or external files are needed.
 
jjcube.js A drop-in Javascript replacement for the PHP Rubik's Cube image generator. The advantage of this Javascript library is that all images are generated by the browser, rather than needing to be generated on the (PHP) server.

twedra.min.js A (50 Kb) minified version of twedra.js.
sandbox A sandbox for experimenting with twedra.js configuration.

Many of the pages above contain virtual cubes which are used to illustrate steps of the solution. The virtual cubes can be interacted with. Clicking and dragging on the virtual cube will turn a face. Clicking and dragging outside the virtual cube will rotate the cube. Below most virtual cubes is a row of buttons which control the cube.

  • PNG Download a PNG image of the cube.
  • Init Reset the cube to its original state.
  • Prev Move to the previous step.
  • Play Play all remaining steps without stopping. Click again to pause playback.
  • Next Move to the next step.

Some buttons have a counter to their right which displays the current step and the total number of steps.

The Mastermorphix, Axis Cube, Windmill Cube, Crazy Windmill Cube, Mirror Cube, Ghost Cube, Pentacle Cube, Fisher Cube, Crazy Fisher Cube, Barrel Cube, Void Cube, Case Cube, Rubix Impossible are all shape-mods of a 3x3x3, so, in theory, they can all be solved using any method that can solve a 3x3x3. The Rex Cube is exactly the same as the Super Ivy Cube which is a shape-mod of the FTO. The Skewb and the Jing Pyraminx can be solved with the use of SledgeHammers.
